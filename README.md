# Goldberg

Goldberg is a Discord bot built for game development teams. He connects to your SVN repository, tracks commits, and helps manage your team's workflow, all with a bit of attitude.

## Features

- **SVN Commit Notifications** — Goldberg polls your SVN repo and posts a message in a designated channel every time a new commit is made.
